## 📋 Project Overview

This project is a modern, feature-rich web application that showcases innovative solutions using web technologies. The application is built with a **frontend-backend architecture**, allowing for scalability and maintainability.

### Key Features
- ✨ Modern, responsive user interface

- 🎯 User-friendly experience

- 🚀 Deployed and production-ready

## 🚀 Getting Started

### Prerequisites
- Git
- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Regurikarthik14/Odoo_hackthon_project.git
   cd Odoo_hackthon_project
   ```

2. **Setup Frontend:**
   ```bash
   cd frontend
   npm install
   npm start
   ```

3. **Setup Backend:**
   ```bash
   cd backend
   npm install
   npm run start
   ```

4. **Access the application:**
   - Frontend: http://localhost:3000 (or your configured port)
   - Backend API: http://localhost:5000 (or your configured port)

## 📝 Configuration

Create a `.env` file in both frontend and backend directories with necessary environment variables:

**Frontend .env:**
```
REACT_APP_API_URL=http://localhost:5000
```

**Backend .env:**
```
PORT=5000
DATABASE_URL=your_database_url
```

## 🧪 Testing

To run tests:


## 📦 Build & Deployment

### Build Frontend:
```bash
cd frontend
npm run build
```

### Deploy to Vercel:
```bash
vercel deploy
```

For detailed deployment instructions, refer to the [Vercel documentation](https://vercel.com/docs).
